# chlookup
Cantonese and Mandarin Pronunciation Lookups from the Command-Line

### Example:

<code>$ python chlookup/chlookup.py 鄧小平</code>

yields:

```
======== 真好笑 牛上樹 ========
+----+---------+---------+
|    | Canto   | Mando   |
+====+=========+=========+
| 鄧 | dang6   | deng4   |
+----+---------+---------+
| 小 | siu2    | xiao3   |
+----+---------+---------+
| 平 | ping4   | ping2   |
+----+---------+---------+
```


The mnemonic <code>真好笑 牛上樹</code> is for memorizing the 6 Cantonese intonations, listed in order.
